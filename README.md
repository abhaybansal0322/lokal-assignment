# Music Player App

## Overview
A React Native music player built using Expo that allows users to search songs, play audio, and control playback via a mini player and full player screen.

## Features
- Search songs
- Play / pause audio
- Mini player
- Full player view
- Artist listing
- Album artwork display

## Tech Stack
- React Native (Expo)
- Zustand (state management)
- Expo AV (audio playback)

## How to Run
1. npm install
2. npx expo start
3. Open in Expo Go on iOS / Android

## Notes
- All data is fetched dynamically from APIs
- No hardcoded songs or artists
